Summary: Latitude vs. X




We first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.


Once the dataset was assembled, Matplotlib was used to plot various aspects of weather vs. latitude. Factors analyzed included: temperature, cloudiness, wind-speed and humidity.


Moreover, Matplotlib was used in order to generate plots aspects of the weather based on latitude. The goal was to visualize a world map using weather plots. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
